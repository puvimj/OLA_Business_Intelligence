# 🚖 OLA Business Intelligence Dashboard

## 📌 Project Overview

The **OLA Business Intelligence Dashboard** is an end-to-end data analytics project developed to analyze ride-booking data and derive actionable business insights. The project integrates **SQL**, **Python**, **Streamlit**, and **Power BI** to provide an interactive analytics platform for exploring booking trends, cancellations, revenue, customer ratings, and ride performance.

The application allows users to execute SQL queries, visualize KPIs, and interact with dashboards through a user-friendly interface.

---

## 🎯 Objectives

* Analyze OLA ride booking data.
* Identify booking and cancellation patterns.
* Evaluate revenue and ride performance.
* Compare vehicle-wise business metrics.
* Build an interactive dashboard for business users.
* Demonstrate SQL, Power BI, and Streamlit integration.

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/OLA_Business_Intelligence.git
```

### 2. Navigate to the project

```bash
cd OLA_Business_Intelligence
```

### 3. Create a virtual environment

```bash
python -m venv venv
```

### 4. Activate the virtual environment

**Windows**

```bash
venv\Scripts\activate
```

### 5. Install dependencies

```bash
pip install -r requirements.txt
```

### 6. Run the Streamlit application

```bash
streamlit run olaApp.py
```
